# API Test Flow

File nay tao ngu canh test cho CRUD nen tang cua chat app hien tai.

Base URL:

```text
http://localhost:8080
```

Chay app local:

```powershell
cd src
.\mvnw.cmd spring-boot:run
```

Neu da test truoc do va bi trung `username`/`email`, hay doi suffix `alice_test`, `bob_test`, `charlie_test` thanh ten moi.

## Ngu Canh

Tao 3 user:

```text
Alice
Bob
Charlie
```

Sau do test flow:

```text
1. Alice va Bob tao private conversation
2. Alice gui message cho Bob
3. Bob gui message lai Alice
4. Lay danh sach message cua conversation
5. Sua message
6. Mark message da doc
7. Xoa mem message
8. Tao group conversation
9. Them Charlie vao group
10. Charlie gui message trong group
```

## User API

### Create Alice

```http
POST /api/user
Content-Type: application/json
```

```json
{
  "username": "alice_test",
  "password": "123456",
  "email": "alice_test@example.com"
}
```

### Create Bob

```http
POST /api/user
Content-Type: application/json
```

```json
{
  "username": "bob_test",
  "password": "123456",
  "email": "bob_test@example.com"
}
```

### Create Charlie

```http
POST /api/user
Content-Type: application/json
```

```json
{
  "username": "charlie_test",
  "password": "123456",
  "email": "charlie_test@example.com"

```

### Get Users

```http
GET /api/user
```

## Conversation API

### Create Private Conversation

Thay `1`, `2` bang `id` cua Alice va Bob.

```http
POST /api/conversations
Content-Type: application/json
```

```json
{
  "type": "PRIVATE",
  "name": null,
  "imgUrl": null,
  "memberIds": [1, 2]
}
```

### Get Conversations

```http
GET /api/conversations
```

### Get Conversation Detail

```http
GET /api/conversations/1
```

### Create Group Conversation

```http
POST /api/conversations
Content-Type: application/json
```

```json
{
  "type": "GROUP",
  "name": "Test Group",
  "imgUrl": null,
  "memberIds": [1, 2]
}
```

### Add Charlie To Group

Thay `2` bang `id` cua group conversation, va `3` bang `id` cua Charlie.

```http
PUT /api/conversations/2
Content-Type: application/json
```

```json
{
  "name": "Test Group",
  "imgUrl": null,
  "addMemberIds": [3],
  "removeMemberIds": []
}
```

## Message API

### Send Message

Thay `1` bang `conversationId`, `senderId` bang user id cua nguoi gui.

```http
POST /api/conversations/1/messages
Content-Type: application/json
```

```json
{
  "senderId": 1,
  "content": "Hello Bob"
}
```

### Get Messages

```http
GET /api/conversations/1/messages
```

### Update Message

Thay `1` bang `messageId`.

```http
PATCH /api/messages/1
Content-Type: application/json
```

```json
{
  "content": "Hello Bob, edited"
}
```

### Mark Message As Read

```http
PATCH /api/messages/1/read
```

### Soft Delete Message

```http
DELETE /api/messages/1
```

## Negative Tests

### Tao Trung Private Conversation

Neu tao lai private conversation voi cung 2 user, API nen tra `409 Conflict`.

```http
POST /api/conversations
Content-Type: application/json
```

```json
{
  "type": "PRIVATE",
  "name": null,
  "imgUrl": null,
  "memberIds": [1, 2]
}
```

### User Khong Thuoc Conversation Gui Message

Neu Charlie khong nam trong private conversation cua Alice va Bob, API nen tra `400 Bad Request`.

```http
POST /api/conversations/1/messages
Content-Type: application/json
```

```json
{
  "senderId": 3,
  "content": "I should not be allowed here"
}
```

## Dau Hieu Flow Dung

Ban biet flow dung khi:

```text
Create user tra ve UserResponse co id
Create conversation tra ve ConversationResponse co members
Send message tra ve MessageResponse co conversationId va senderId
Get messages tra ve danh sach message theo createdAt tang dan
Update message tra ve edited = true
Mark read tra ve readAt khac null
Delete message tra ve HTTP 200/204 va message bi set deleted = true trong database
```

Co the chay nhanh bang file:

```text
src/api-test-flow.http
```
